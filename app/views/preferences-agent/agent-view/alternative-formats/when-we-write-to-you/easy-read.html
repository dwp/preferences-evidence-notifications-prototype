Tell the customer about Easy Read

Easy Read uses simple language and images to make information easier to understand. It is designed to help those with learning difficulties.

Easy Read is not normally appropriate for general communication. It should only be used if you really believe no other alternative format would be a better option.

Discuss with the customer their needs and why they want to choose Easy Read. Offer alternatives and only select Easy Read if it really is the best option for them.
