Baseline

Based on a Discovery and two rounds of UR, we are happy with this baseline prototype. The intension is for the developers to start building it and for the UCD team to iterate as the requirements develop further and we learn more about the technical limitations, such as Akamai. Also major new changes will be documented as a change request.

Main error messaging

  • Select Yes if you have been contacted to send us evidence
  • Select Yes if you are sending us evidence for Employment and Support Allowance (ESA)
  • Select which evidence you are sending
  • You must upload at least one file to continue
  • Select Yes if you want to upload another [evidence type] file
  • Select Yes if you want to submit these files

Upload error messaging scenarios

  • You must upload at least one file to continue
  • The selected file must be a JPG, JPEG, HEIC, HEIF, BMP, PNG, TIF or PDF
  • The selected file must be smaller than 10MB
  • The selected file is empty
  • The selected file contains a virus
  • The selected file is password protected
  • The selected file could not be uploaded - try again

Sign out

  • You have signed out (action performed by customer)
  • You have been signed out
  • Please wait while we sign you out

Service unavailability or problems

  • There has been an error when uploading your files
  • You'll need to start again
  • There is a problem with the service page
  • Service unavailable page
